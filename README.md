# Fire Detection — YOLOv8 + D-Fire Dataset

A computer vision project for detecting fire and smoke in images using YOLOv8, trained on the D-Fire dataset.

## Setup

```bash
# Install dependencies
uv sync
```

## Dataset

This project uses the [D-Fire dataset](https://github.com/gaiasd/DFireDataset). Download and place the dataset contents inside the `data/` directory.

## Usage

```bash
# Run scripts from the scripts/ directory
python scripts/<script_name>.py
```
